# 坦克大戰 (Tank Battle Game)

一個基於 HTML5 Canvas 和 JavaScript 開發的現代化坦克大戰遊戲。

## 🎮 遊戲特色 (Features)

### 核心體驗 (Core Experience)
*   **專業級遊戲引擎**：實現了 Delta Time (獨立幀率) 和 AABB 物理碰撞系統，確保流暢的操作手感。
*   **自適應畫面**：支援 PC 和行動裝置，畫面可自動適應螢幕比例。
*   **豐富的關卡**：內建 **10 個精心設計的關卡**，包含不同地形與戰術挑戰。
    *   Level 1-5: 基礎訓練與迷宮挑戰。
    *   Level 6: 叢林伏擊 (隱蔽戰)。
    *   Level 7: 死亡迴廊 (狹窄通道)。
    *   Level 8: 破碎島嶼 (水域概念)。
    *   Level 9: 棋盤戰場 (密集障礙)。
    *   Level 10: 最終防線 (堡壘攻堅)。

### 武器系統 (Weapon System)
*   **普通砲彈**：標準攻擊。
*   **散彈槍 (Shotgun)**：拾取道具 'W' 獲得，一次發射 3 發子彈，適合近戰爆發。
*   **雷射 (Laser)**：拾取道具 'Z' 獲得，高速彈道 (未來可升級穿透)。

### 操作方式 (Controls)

#### 電腦版 (Desktop)
*   **移動**: `WASD` 或 `方向鍵`
*   **射擊**: `空白鍵 (Space)`
*   **暫停**: 點擊按鈕或 `P` (待實作)
*   **AI 狀態開關**: `I`

#### 手機版 (Mobile)
*   **虛擬搖桿**: 螢幕左下方十字鍵控制移動。
*   **射擊按鈕**: 螢幕右下方按鈕射擊。
