## 🎮 遊戲特色 (Features)

### 核心體驗 (Core Experience)
*   **專業級遊戲引擎**：實現了 Delta Time (獨立幀率) 和 AABB 物理碰撞系統，確保流暢的操作手感。
*   **自適應畫面**：支援 PC 和行動裝置，畫面可自動適應螢幕比例。
*   **豐富的關卡**：內建 **10 個精心設計的關卡**，包含不同地形與戰術挑戰。

### 武器系統 (Weapon System)
*   **普通砲彈**：標準攻擊。
*   **散彈槍 (Shotgun)**：拾取道具 'W' 獲得，一次發射 3 發子彈，適合近戰爆發。
*   **雷射 (Laser)**：拾取道具 'Z' 獲得，高穿透性彈道。

### 操作方式 (Controls)

#### 電腦版 (Desktop)
*   **移動**: `WASD` 或 `方向鍵`
*   **射擊**: `空白鍵 (Space)`
*   **暫停**: `P` (待實作)
*   **AI 狀態開關**: `I`

#### 手機版 (Mobile)
*   **虛擬搖桿**: 螢幕左下方十字鍵控制移動。
*   **射擊按鈕**: 螢幕右下方按鈕射擊。
